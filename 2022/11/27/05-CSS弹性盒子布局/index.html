CSS弹性盒子布局

CSS 弹性盒子布局是 CSS 的模块之一，定义了一种针对用户界面设计而优化的 CSS 盒子模型。

这是一种用于在单个维度（行或列）中显示项目的布局模型

  • 在弹性布局模型中，弹性容器的子元素可以在任何方向上排布，
  • 也可以“弹性伸缩”其尺寸，既可以增加尺寸以填满未使用的空间，也可以收缩尺寸以避免父元素溢出。
  • 子元素的水平对齐和垂直对齐都能很方便的进行操控。
  • 通过嵌套这些框（水平框在垂直框内，或垂直框在水平框内）可以在两个维度上构建布局。

简单例子

  • 外层div设置 display:flex 成为弹性容器
  • 内层三个div称为弹性项
  • 内层三个div自动沿着弹性盒子主轴方向（当前默认为水平方向）排列

主轴

在弹性容器中使用flex-direction;属性可以修改主轴方向

  • row ：水平方向
  • row-reverse ：反水平方向：即向左为正，元素从容器最右侧开始排列
  • column ：垂直方向
  • column-reverse

  • 设置flex-direction:row-reverse
  • 从最右侧开始水平排列 1的div元素在最右侧

交叉轴（侧轴）

交叉轴的方向与主轴垂直

若主轴为 row 或 row-reverse

则侧轴为 column （垂直方向）

align-items属性：align-items属性设置弹性项在弹性容器交叉轴方向上的对齐方式


弹性盒子的属性

  • flex-direction:row-reverse/row/column/column-reverse：设定主轴方向

  • flex-wrap: nowrap/wrap：设定子元素总宽度超过父容器宽度时的换行与否

    • nowrap不换行，缩进超长元素，不能缩进则会溢出
    • wrap超长元素换到下一行显示，每一行相当于一个flex容器

  • flex-flow：flex-direction 和 flex-wrap的简写，指定两个属性的值

  • align-item：规定弹性项在侧轴方向上的排列方式

    • stretch：默认值，所有弹性项都会被拉伸到与最高弹性项相同的高度
    • flex-start：顶部对齐
    • flex-end：底部对齐
    • center：在侧轴上居中排列

  • justify-content：规定弹性项在主轴上的排列方式

    • stretch
    • flex-start：默认值，从容器起始线开始排列
    • flex-end：从容器终止线开始排列
    • center：主轴居中排列
    • space-around：元素之间间隔相等
    • space-between：每个元素的左右空间相等

弹性项（弹性盒子中的子元素）的flex属性

flex属性是三个属性的简写

  • flex-grow 默认值为0
  • flex-shrink 默认值为1
  • flex-basis 默认值为auto

单值语法: 值必须为以下其中之一:

  • 一个无单位数: 它会被当作flex:<number> 1 0; <flex-shrink>的值被假定为1，然后<flex-basis> 的值被假定为0。
  • 一个有效的**宽度(width)**值: 它会被当作 <flex-basis>的值。
  • 关键字none，auto或initial.

双值语法: 第一个值必须为一个无单位数，并且它会被当作 <flex-grow> 的值。第二个值必须为以下之一：

  • 一个无单位数：它会被当作 <flex-shrink> 的值。
  • 一个有效的宽度值: 它会被当作 <flex-basis> 的值。

三值语法:

  • 第一个值必须为一个无单位数，并且它会被当作 <flex-grow> 的值。
  • 第二个值必须为一个无单位数，并且它会被当作 <flex-shrink> 的值。
  • 第三个值必须为一个有效的px值， 并且它会被当作 <flex-basis> 的值。

flex-grow属性

处理弹性项尺寸增长问题

CSS属性 flex-grow设置了一个flex项主尺寸的flex增长系数。它指定了flex容器中剩余空间的多少应该分配给项目（flex增长系数）。

主尺寸是项的宽度或高度，这取决于flex-direction主轴方向。

剩余的空间是flex容器的大小减去所有flex项的大小。如果所有的兄弟项目都有相同的flex-grow系数，那么所有的项目将获得相同的剩余空间，否则将根据不同的flex-grow系数定义的比例进行分配。

flex-shrink属性

处理弹性项尺寸收缩问题

CSS flex-shrink 属性指定了 flex 元素的收缩规则。flex 元素仅在默认宽度之和大于容器的时候才会发生收缩，其收缩的大小是依据 flex-shrink 的值。

flex-basis属性

处理弹性项初始大小问题

若元素设定尺寸，则flex-basis值即为设定的尺寸

若元素没有尺寸，则flex-basis值为内容的尺寸

CSS 属性 flex-basis 指定了 flex 元素在主轴方向上的初始大小。如果不使用 box-sizing 改变盒模型的话，那么这个属性就决定了 flex 元素的内容盒（content-box）的尺寸。
