1.安装

2.基本使用

  • React.createElement() 创建一个React元素

    三个或以上参数

    • 参数1：元素名
    • 参数2：元素属性对象
    • 参数3,4,…：元素子节点

  • ReactDOM.render() 将创建的React元素渲染到页面的容器DOM元素中

    两个参数

    • 参数1：React元素
    • 参数2：作为容器的DOM元素

3.React脚手架

类似于Vue脚手架

npx create-react-app my-app 创建一个名为my-app的react-app

npm start 在react-app目录下执行开启项目


3.1 使用

  • 在项目入口文件(index.js)渲染React元素，容器为public/index.html中的id为root的div元素
  • App.js文件类似于Vue中的App.vue，是React的根元素
  • React和ReactDOM通过ES6模块化引入(项目初始化时自动完成)
