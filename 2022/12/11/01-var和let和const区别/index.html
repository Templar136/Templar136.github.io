01-var/let/const区别

发表于 2022-12-11 分类于 es6

var和let和const区别

var

var 声明变量 没有块级作用域 存在变量提升(将声明提升至函数或代码段顶部) 值可更改

let es6新增

let 声明变量 有块级作用域 不存在变量提升(只能先声明再使用) 值可更改

const es6新增

const 声明常量 有块级作用域 不存在变量提升(只能先声明再使用) 值不可更改

值不可更改(地址不可更改)：

  • 基本数据类型不可更改
  • 引用数据类型可以修改内部数据结构(但不能直接修改整体引用数据，会导致地址变化)

const常量声明时必须初始化，否则报错

当声明的量之后不希望变化（占用的内存地址不变） 最好使用const关键字声明(系统不需要监控常量 提升效率)

let和const在全局声明时不会在window对象中创建属性

就像const 一样，let不会在全局声明时创建window 对象的属性。

**注意: ** Window 和 window是两个对象

  • 输出顶级对象的语句为console.log(window)
  • 调用a函数时，函数中的this确实指向window，window.num为undefined
  • 但是在全局作用域下直接输出num变量可以得到其值为1
