23-JS字符串常用方法

发表于 2022-12-04 分类于 javaScript

字符串常用方法

charAt()方法

charAt() 方法从一个字符串根据索引返回指定的字符。

includes()方法

includes() 方法用于判断一个字符串是否包含在另一个字符串中，根据情况返回 true 或 false。

两个参数

  • searchString

    要在此字符串中搜索的字符串。

  • position 可选

    从当前字符串的哪个索引位置开始搜寻子字符串，默认值为 0

concat()方法

concat() 方法将一个或多个字符串与原字符串连接合并，形成一个新的字符串并返回。

str.concat(str1[,str2,...,strn])

indexOf()方法

indexOf() 方法返回调用它的 String 对象中指定值或子串第一次出现的索引

从 fromIndex 处进行搜索。如果未找到该值或子串，则返回 -1。

fromIndex默认值为0

match()方法 匹配正则

match() 方法检索字符串

以数组形式返回该字符串匹配正则表达式的结果。

如果你没有给出任何参数并直接使用match() 方法 ，你将会得到一 个包含空字符串的 Array ：[“”] 。

substring()方法

substring() 方法返回一个字符串在开始索引到结束索引之间的一个子串, 或从开始索引直到字符串的末尾的一个子集。

  • 包含开始索引，不包含结束索引

replace()方法

将给定模式匹配的子串替换为新字符串

返回值：替换后的新字符串，不修改原字符串

  • regexp(模式)：正则表达式字面量 或 正则表达式对象

  • substr(模式)：一个字符串(str中被替换的子串)

  • newSubstr(替换值)：新字符串

  • function(替换值)：该函数的返回值作为替换值

当模式使用regexp时，所有匹配的子串都会被替换

当模式使用substr时，只有满足的第一个子串被替换

split()方法

将字符串以给定字符为分隔，分隔为数组

返回值：分隔后的数组，不修改原字符串
