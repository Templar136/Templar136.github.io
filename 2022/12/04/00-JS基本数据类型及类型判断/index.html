JS数据类型及类型判断

5种简单数据类型（也称为基本数据类型）: Undefined、Null、Boolean、Number和String

1种复杂数据类型：Object

  • Object是所有对象的父类
  • 引用类型又可以细分为 Array Function Object

js数据类型

  • Number（数字）

  • String（字符串）

  • Boolean（布尔）

  • Symbol（符号）（ES2015 新增）即（ES6新增）

  • Object

    （对象）

    • Function（函数）
    • Array（数组）
    • Date（日期）
    • RegExp（正则表达式）

  • null（空）

  • undefined（未定义）


1. Number

JavaScript的Number类型为双精度IEEE 754 64位浮点类型。

也就是说可以表示整数和小数

Number对象的方法

  • Number.isNaN()

    • 判断参数是否为NaN

  • Number.isInteger()

    • 参数是否为数字且为整数

  • Number.parseInt()

    • 与全局对象parseInt()相同
    • parseFloat()同理

Number()构造器返回值

使用Number函数将参数转换为数字

  • 对于有小数点但小数位均为0的浮点数，返回整数
  • 对于空字符串和null，返回0
  • 对于非数值（非空字符串），返回NaN (NaN的类型为Number)
  • NaN与任何数据都不相等 返回false（包括NaN）
  • 对于字符串 ‘-Infinity’ 和 ‘Infinity’，返回其本身

2. String

字符串字面量采取以下方式表示：

  • ‘hello world’单引号
  • "hello world" 双引号
  • `hello ${a}` 反引号
    • ES6新增的模板字符串表示方式
    • 可以使用${变量名}将变量值直接拼接在字符串中，不需要使用普通字符串的+拼接形式
    • ${a} 即将a变量的值比如’world’拼接进字符串

转义字符


获取单个字符

  • 使用charAt('字符串索引')方法
  • 将字符串当作一个类似数组的对象，使用数值索引获取其中单个字符

字符串比较

直接使用比较运算符（<,>,=等）

比较时使用字符的ASCⅡ码进行比较


3. Boolean

对于Boolean()构造器

将参数转换为Boolean值

  • b的类型为Boolean
  • b的值为false

对于new Boolean()

其返回一个Boolean对象，注意是对象

  • a为一个Boolean对象，值为Boolean类型的false
  • 判断时将a当作对象，非空对象判断时返回true

4. Symbol

symbol 是一种基本数据类型 （primitive data type）。Symbol()函数会返回symbol类型的值，该类型具有静态属性和静态方法。它的静态属性会暴露几个内建的成员对象；它的静态方法会暴露全局的symbol注册，且类似于内建对象类，但作为构造函数来说它并不完整，因为它不支持语法：”new Symbol()“。

  • 每个从Symbol()返回的symbol值都是唯一的。（即使参数一样，返回的值也不同）
  • 一个symbol值能作为对象属性的标识符；这是该数据类型仅有的目的。

5.null

JavaScript的基本数据类型之一

null指空对象或对象的值未设置，在布尔运算中被认为是false

  • null不是全局对象（window）的一个属性，它指示变量未指向任何对象

null与undefined的不同点

  • typeof null 返回object（null是空对象）

  • typeof undefined 返回 undefined

  • ===不执行类型转换 所以null === undefined // false

  • ==执行类型转换 null == undefined // true

  • isNaN(1 + null) // false

  • isNaN(1 + undefined) // true


6. undefined

它是一个JavaScript的 原始数据类型 。

  • 一个被声明但是没有被赋值的变量的类型是undefined

  • 一个函数如果没有使用return语句指定返回值，就会返回一个undefined值。

  • 一个未赋值的变量 === undefined 时返回true
  • typeof 一个未赋值的变量 === 'undefined' 时返回true
    • typeof返回一个代表类型的字符串

类型判断

typeof

typeof 操作符返回一个字符串，表示未经计算的操作数的类型。

即typeof 操作符返回的值为字符串，字符串再经typeof操作符后返回String

typeof使用时两种方式：

  • typeof a
  • typeof (a)

typeof操作符常见的返回值

  • typeof null 返回’object’
  • typeof function对象 返回’function’

new操作符生成的实例对象 使用typeof时的返回值

  • 使用new操作符生成的实例对象本质是对象，所以使用typeof返回’Object’,
  • 但有一个特例 new Function 时产生的实例对象 使用typeof时返回’function’

instanceof

使用方法：

instanceof 运算符用来检测 constructor.prototype 是否存在于参数 object 的原型链上。

  • 实例对象o的原型链为 o -> C.prototype -> Object.prototype -> null
    • 所以o instanceof C 返回true
    • o instanceof D 返回false
