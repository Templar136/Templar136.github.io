24-JS数组常用方法

发表于 2022-12-04 分类于 javaScript

数组常用方法

forEach(callback)

遍历数组，对数组每个元素执行forEach方法的回调函数

slice()方法

slice() 方法返回一个新的数组对象，

这一对象是一个由 begin 和 end 决定的原数组的浅拷贝（包括 begin，不包括end）。

原始数组不会被改变。

  • end省略时，从begin开启浅拷贝后面的所有数组元素

注意：使用slice方法进行浅拷贝时

  1. 对于原数组中的Number和String和Boolean类型数据进行完全拷贝
    • 修改新数组的Number或String或Boolean类型数据不会影响原数组
  2. 对于原数组中的对象引用或者对象或者数组，拷贝引用到新数组中
    • 修改该对象或数组，原数组和新数组均受到影响
    • 通过新数组中拷贝的引用修改对象或数组，两个数组均受到影响
  3. 向两个数组任一添加新元素，另一个不会收到影响

splice()方法

  • 从start索引开始删除deleteCount个元素，并添加 item1，item2等元素

  • 若deleteCount为0，则不移除元素，在start索引位置开始添加元素，原start索引的元素后移

  • 会修改原数组，

  • 返回值：返回删除元素组成的数组，没有删除元素则返回空数组

reduce()方法

返回累加后的结果

参数：

callback 累计器回调函数

数组中每个值执行的回调函数，包含四个参数：

  • accumulator

    累计值： 它是上一次调用回调时的返回值，或initialValue（见于下方）。

  • currentValue

    数组中正在处理的元素。

  • index 可选

    数组中正在处理的当前元素的索引。 如果提供了initialValue，则起始索引号为0，否则从索引1起始。

  • array可选

    调用reduce()的数组


initialValue可选

  • 不省略：作为第一次调用 callback函数时的accumulator（累计起始值），index从0开始。

  • 省略：则累加起始值默认为数组的第一个元素，index从1开始。

空数组调用且未指定initialValue时，reduce报错

空数组调用但指定initialValue时，返回initialValue


map()方法

map() 方法返回一个新数组，其元素是调用者数组中的每个元素经回调函数处理后的返回值。

返回一个新数组

参数

callback

生成新数组元素的函数，使用三个参数：

  • currentValue

    callback 数组中正在处理的当前元素。

  • index可选

    callback 数组中正在处理的当前元素的索引。

  • array可选

    map 方法调用者数组。

thisArg可选

执行 callback 函数时值被用作this。

shift()方法

将数组首元素移出

返回移出的元素

会修改原数组

unshift()方法

在数组头部加入一个新元素

修改原数组

返回数组长度

pop()方法

删除数组尾元素

返回删除的元素

filter()方法

Array.filter(callback())

对调用者数组的每个元素执行回调函数，通过回调函数(return true)的元素加入新数组

返回生成的新数组 (通过过滤的元素形成的新数组)

不修改原数组

callback回调函数中三个参数

  • currentValue

    callback 数组中正在处理的当前元素。

  • index可选

    callback 数组中正在处理的当前元素的索引。

  • array可选

    filter方法调用者数组。

concat()方法

连接两个数组

arr.concat(arr1)连接数组arr和arr1

返回生成的新数组

不修改原数组

every()方法

数组元素全部都满足回调函数，返回true

否则返回false

some方法

数组元素至少有一个满足回调函数，返回true

否则返回false

join()方法 数组转字符串

返回转换后的字符串

不修改原数组

  • 方法传入空字符串作为参数，将数组每个元素直接拼接为字符串
  • 不传参时，将中括号里的内容转换为字符串

includes()方法

判断一个数组是否包含一个指定的值，如果包含则返回 true，否则返回 false。


sort()方法

根据Array.sort(compareFunction)的 compareFunction函数确定排序规则

compareFunction(a,b)

  • a和b为比较的两个元素
  • 函数返回值 < 0 时，a放在b前
  • 函数返回值 > 0 时，b放在a前
  • 函数返回值 = 0 时，ab位置不变

返回排序后的数组。（注意：数组已原地排序，并且不进行复制。）

  • 对于数字数组nums，进行升序排序，修改原数组，返回排序后的数组
