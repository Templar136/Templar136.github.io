32-JS中valueOf和toString区别

发表于 2022-12-04 分类于 javaScript

valueOf和toString方法

valueOf方法

返回指定对象的原始值

toString()方法

返回一个表示该对象的字符串

  • object.toString()：默认返回’[object type]‘，type为对象的类型，取值为Date、Object等

    • 可以覆写对象的toString()方法来返回其他值

    • ```js
      function Dog(name) {
      this.name = name;
      }

      const dog1 = new Dog(‘Gabby’);

      Dog.prototype.toString = function dogToString() {
      return this.name;
      };

      console.log(dog1.toString());
      // expected output: “Gabby”

  • Date.toString()：返回时间对象在当前时区的字符串格式表示
