CSS盒模型

发表于 2022-11-27 分类于 HTMLandCss

CSS盒模型

当一个文档进行布局时浏览器渲染引擎会根据设定的盒模型将所有元素表示为一个矩形区域

  • 行内元素的矩形区域没有width和height等值

CSS3中的box-sizing属性可以指定页面元素使用的盒模型

  • box-sizing属性定义了如何计算一个元素的总高度和总宽度

box-sizing的属性取值

  • content-box：默认值，
    • width和height只包含内容区域的宽高 width=内容宽度
    • 不包含margin，padding和border
    • margin，padding和border都在盒子外部
  • border-box：
    • width = 内容宽度 + padding + border
    • 宽度和高度不包含 margin
  • inherit：从父元素继承box-sizing的值
    • 此属性MDN文档没有提及

CSS弹性盒子布局

发表于 2022-11-27 分类于 HTMLandCss

CSS弹性盒子布局

CSS 弹性盒子布局是 CSS 的模块之一，定义了一种针对用户界面设计而优化的 CSS 盒子模型。

这是一种用于在单个维度（行或列）中显示项目的布局模型

  • 在弹性布局模型中，弹性容器的子元素可以在任何方向上排布，
  • 也可以“弹性伸缩”其尺寸，既可以增加尺寸以填满未使用的空间，也可以收缩尺寸以避免父元素溢出。
  • 子元素的水平对齐和垂直对齐都能很方便的进行操控。
  • 通过嵌套这些框（水平框在垂直框内，或垂直框在水平框内）可以在两个维度上构建布局。
